# Desafio MV - Frontend


### Framework
* Angular / Typescript

### Acesse 
## http://desafiomv.ourmind.com.br/login

#### Desafio  
* Desenvolver uma aplicação que realize cadastro e visualização de estabelecimentos e profissionais.

#### Atenção
* Configurar o env corretamente com o backend

### Rodar a aplicação
```
npm install
ng server --port 4200
```
